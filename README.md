# DogWebsiteUsingJS
This project is a multi-page web application built using vanilla JavaScript, HTML, and CSS.
It demonstrates core front-end concepts including DOM manipulation, Browser Object Model (BOM), cookies, timers, and API integration.

The application allows users to:

Sign up and log in using cookies

View a home page with an automatic image slider

Browse dog images loaded dynamically from a public API

Navigate between pages and view image details
